# oxProbs

oxProbs analyze the data of an OXID eShop for finding problems like
* Articles 
   * without a stock
   * without a price
   * without a category
   * with deactivated parents (orphaned variants)
* Categories
   * with deactivated parents

and many other unwanted data.
